## Overview

The Call Logs page in ScheduleTime provides a detailed view of all call activities associated with AI agents. This feature allows users to track and analyze call data for better management and insights.

## Accessing the Page

1. Log in to your ScheduleTime account.
2. Navigate to the ScheduleTime dashboard.
3. Click on the `Data` tab in the left-hand navigation menu, then select `Call Logs` from the submenu.

## Call Logs Interface

### Agent Selection

Use the "Select Agent" dropdown at the top of the page to filter logs by specific AI agents.

### Display Options

- Adjust the number of entries per page using the dropdown (default: 10 entries).
- Use the search bar in the top right to find specific log entries.

### Call Log Table

The table displays the following columns:

- **Call ID:** Unique identifier for each call
- **Connector:** The connection method used for the call
- **Agent:** The AI agent handling the call
- **User:** The user associated with the call
- **Time Usage:** Duration of the call
- **Status:** Current status of the call
- **To Number:** The recipient's phone number
- **From Number:** The originating phone number

*Note: If no call logs are available, the message "No entries found" is displayed.*

### Additional Features

- **Sorting:** Click on column headers to sort the data.
- **Filtering:** Use the search function to filter logs based on specific criteria.

## Important Notes

- Ensure you have the necessary permissions to access call log data.
- Call logs provide valuable insights into AI agent performance and user interactions.
- Regularly reviewing call logs can help optimize AI agent configurations and improve overall service quality.
