## Overview

The Outgoing Agent allows users to initiate outgoing calls by adding a phone number to dial. This agent is suitable for making individual outgoing calls.

## Steps to Initiate an Outgoing Call

1. **Access the Outgoing Agent**:
   - Navigate to the `Manage Agents` interface in your system.
   - There will be agents of 3 types in the list - Incoming, Outgoing and Batch.
   - Click on the `Invoke` button for Outging Agent which you want to use for your call.

2. **Enter Phone Number**:
   - A prompt will appear asking you to input the phone number to dial.

3. **Dial the Number**:
   - Enter the desired phone number in the provided field.

4. **Initiate the Call**:
   - Once the number is entered, confirm to initiate the outgoing call.

This process enables users to efficiently make outgoing calls through the Outgoing Agent, ensuring a straightforward calling experience.
