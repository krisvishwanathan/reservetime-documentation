## Overview

Knowledge Agents in ScheduleTime are AI-powered assistants that allow you to upload PDF files and interact with them by asking questions. These agents extract and analyze information from the PDFs, enabling you to quickly find answers and insights within the documents.

## Accessing the Knowledge Agents Interface

1. Log in to your ScheduleTime account.
2. Navigate to the ScheduleTime dashboard.
3. Click on the `Agents` section in the left-hand navigation menu and select `Knowledge Agents`.

## Adding a New Knowledge Agent

To add a new agent, click the `Add Agent` button on the Knowledge Agents page.

### Agent Creation Form

Upon clicking `Add Agent`, a form will appear. Fill out the required details as follows:

1. **Agent Name:** Enter a name for the agent.
2. **Model:** Select the AI model (e.g., gpt-3.5-turbo-0125).
3. **API Key:** Enter your API key for the selected model.
4. **Documents:** Upload the PDF files you want the agent to analyze.

### Starting a Chat

1. On the Knowledge Agents page, locate the agent you want to interact with.
2. Click on `Chat` option from the three-dot menu.
3. This will open the chat interface for the selected Knowledge Agent.

### Using the Chat Interface

1. Type your question or query in the chat input field.
2. Press Enter or click the Send button to submit your question.
3. The Knowledge Agent will process your query and provide a response based on the information in the uploaded documents.
4. You can ask follow-up questions or new queries to explore the document content further.

### Knowledge Agent

- **Description:** The Knowledge Agent processes uploaded PDF documents and answers questions based on their content.

- **Usage:** Ideal for scenarios where you need quick access to information contained in large documents or multiple PDFs.

## Managing Existing Agents

On the Knowledge Agents page, you can view, edit, or delete existing agents. Each agent is displayed with its name and associated model.

- **View Options:** Switch between different views (grid or list) for a better overview of your agents.

- **Edit:** Click the three dots next to an agent to modify the agent's details or upload additional documents.

- **Delete:** Click the three dots next to an agent to remove the agent and its associated documents.

## Agent Functionality

Knowledge Agents are designed to:

1. Process and analyze uploaded PDF documents.
2. Extract key information and insights from the documents.
3. Answer questions based on the content of the uploaded PDFs.
4. Provide quick access to specific information within large documents.
5. Handle complex queries and provide relevant, accurate responses.
